按钮回显功能

技术 2022/6/26

keep-alive的使用

  • 在写一个点击按钮进入详情之后还能回到之前选择的按钮的功能，刚好之前看文档的时候记得有一个keep-alive 就尝试了一下。
  • 使用Keep-alive 来保持页面的状态，再配合beforeRouteLeave 判断页面是否进行保持状态
  • 在列表页上的其他可跳转页面的按钮，比如 编辑按钮等 加上 钩子函数beforeRouteLeave（回到列表页不缓存列表页面，即刷新）
  • 目前是这么写的，以后遇到新的方法再来添加。

Node和node-Sass 版本兼容问题

工具 2022/6/25

node和node-sass 版本兼容问题

这是一个朋友遇到的问题，搞半天都不行，然后来问我。我也是搞了半天才给解决。
  • 首先删除本地的node所有文件（建议是Geek卸载）
  • 其次下载nvm中叫nvm-setuo.zip，直接解压后 点击exe程序 进行安装 安装之后。
  • 打开CMD（一定要管理员权限）输入nvm v 查看是否有版本号。之后输入nvm install 版本号 即可安装 nvm use 版本号 之后进行测试 node -v和 npm -v
  • 这样就可以选择当前node-sass对应的版本号，同时在遇到其他项目时，还可以切换其他node版本号。

Git的使用

工具 2022/6/22

从gitlab上面拉取代码以及上传

  • 公司项目都是部署在gitlab上面，
  • gitlab使用起来和github差不多，
  • 负责的项目就我和另外一个同事写，我两基本就是写完直接推main
  • 总结下来的操作：
  • git add . 添加之前全部的修改
  • git commit -m 'xxx' 提交到本地仓库
  • git pull origin 远端分支 一般我都是直接问有没有改动 有就先拉取
  • git push origin HEAD:远端分支名 在没有冲突的情况下推送到远程仓库

Hexo-Blog搭建

技术 2022/6/12
  • 搭建这个blog，使用了Hexo这个开箱即用的博客框架，配合GithubPages,算是一种白嫖的方式。
    • 为了加快访问速度，网站同样也托管在了vercel平台上面，图片基本使用了白嫖的图床(https://imgtu.com/)。
    • 整个框架使用起来还蛮容易上手，也参考了网上很多的教程。有一些常用的命令：
      • 生成静态文件：hexo g
      • 清除静态文件：hexo cl
      • 在本地运行：hexo s
      • 部署到网站：hexo d
      • 生成静态文件并部署到网站：hexo d -g 或 hexo g -d
      • 创建新文章：hexo new 为文件名
